# ImageFilterApp

Приложение для медианной фильтрации изображений с возможностью добавления импульсного шума.

## Возможности

- Загрузка изображений (BMP, JPG, PNG, GIF, TIFF)
- Добавление шума "соль и перец"
- Классическая и взвешенная медианная фильтрация
- Масштабирование (колесико мыши)
- Сохранение результатов

## Технологии

- .NET Framework / .NET Core
- WPF
- C# 7.3

## Установка

1. Клонируйте репозиторий
2. Откройте `ImageFilterApp.sln` в Visual Studio
3. Соберите проект (Build → Build Solution)

## Использование

1. Загрузите изображение
2. Добавьте шум (по желанию)
3. Примените фильтр
4. Сохраните результат

## Горячие клавиши

- `O` - оригинал
- `N` - с шумом
- `R` - восстановленное
- `+/-` - масштаб
- `Ctrl+F` - подогнать под размер
